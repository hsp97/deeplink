Live Editor 소개

HTML, CSS, JavaScript를 테스트할 수 있는 간단한 코드 플레이그라운드입니다
미리보기는 보안을 위해 샌드박스된 iframe에서 실행됩니다. JavaScript의 콘솔 로그는 아래 콘솔 패널에 캡처되어 표시됩니다.
빠른 프로토타이핑, 페이로드 테스트, 또는 웹 개발 학습에 활용하세요.

  • 1
    좌측 메뉴에서 리스트를 추가해서 코드 입력 페이지를 추가하세요
    이름을 더블클릭하여 리스트 이름을 수정할 수 있고 드래그를 해서 메뉴 순서 변경이 가능합니다
  • 2
    세팅된 버튼
    세팅된 버튼을 누르면 간단한 예시 코드가 세팅 됩니다.
  • 3
    프레임워크 세팅
    필요한 라이브러리를 체크해서 불러올 수 있습니다.
  • 4
    코드 공유
    자신이 작성한 코드를 URL 복사만으로 타인과 공유할 수 있습니다.
  • 5
    html 다운로드
    현재 띄워놓은 리스트의 코드를 다운로드 받을 수 있습니다.
